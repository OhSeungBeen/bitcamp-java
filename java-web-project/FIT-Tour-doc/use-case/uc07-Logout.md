# UC07 - 로그아웃(LogOut)
회원이 로그아웃을 하기위해 사용하는 유스케이스

## 주 액터(Primary Actor)
- 회원
- 관리자
## 보조 액터(Secondary Actor)
- 없음

## 사전 조건(Preconditions)
- 엑터가 로그인 되어 있는 상태이다.

## 종료 조건(Postconditions)
-로그아웃 하였다.

## 시나리오(Flow of Evnets)

### 기본 흐름(Basic Flows)

- 1. 액터가 '로그아웃' 버튼을 클릭할 때 이 유스케이스를 시작한다.
- 시스템은 로그아웃 후 '메인 페이지'를 출력한다.
